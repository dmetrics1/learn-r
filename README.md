# R: de cero a decisiones · _R: From Zero to Decisions_

Guía interactiva y **bilingüe (español / inglés)** para aprender **R** desde cero hasta un nivel profesional de ciencia de datos: análisis, `tidyverse`, visualización con `ggplot2`, estadística, Machine Learning con `tidymodels`, Shiny y despliegue.

> _An interactive, bilingual guide to learning R from scratch to a professional data-science level._

## 🗺️ Contenido / Contents

- **Nivel 1 · Básico** (15 módulos) — instalación, tipos de datos, vectores, data frames, importación y funciones → _Proyecto: Análisis Exploratorio (EDA)_.
- **Nivel 2 · Intermedio** (18 módulos) — programación, `tidyverse`, `dplyr`, `ggplot2`, estadística, Quarto, SQL y dashboards → _Proyecto: Dashboard_.
- **Nivel 3 · Avanzado** (18 módulos) — programación funcional, `data.table`, Machine Learning, series de tiempo, Shiny, Git y despliegue → _Proyecto: Data Science de extremo a extremo_.

Cada módulo sigue una metodología progresiva: **Concepto → Ejemplo → Práctica → Reto → Proyecto**, con código ejecutable, errores frecuentes y ejercicios con solución.

## ⚙️ Tecnología / Tech

Una **SPA (Single Page Application) autocontenida** por idioma: todo el HTML, CSS y JavaScript viven en un solo archivo. Sin servidor ni dependencias que instalar (solo carga tipografías y el resaltador de sintaxis desde CDN). Incluye buscador interno, modo claro/oscuro, índice por página, barra de progreso de lectura y diseño responsive.
