Guía completa e interactiva para dominar **R** de verdad: del primer `install.packages()` hasta un proyecto de **Data Science** desplegable. Tres niveles progresivos, **51 módulos**, ejemplos ejecutables, errores frecuentes, ejercicios con solución y **3 proyectos integradores**. Disponible en **español e inglés** con un clic.

## ✨ Qué encontrarás

- **3 niveles progresivos** — Básico → Intermedio → Avanzado, pensados para no perderte nunca.
- **51 módulos temáticos** con teoría breve, código comentado y salida real.
- **Metodología por módulo:** Concepto → Ejemplo → Práctica → Reto → Proyecto.
- **Errores frecuentes** señalados (los que todos cometemos) y cómo evitarlos.
- **Ejercicios con solución** desplegable en cada tema.
- **3 proyectos integradores** de extremo a extremo (uno por nivel).
- **100% código ejecutable**, actualizado a **R 4.6** y **tidyverse 2.0**.
- **Bilingüe ES/EN**, modo claro/oscuro, buscador interno y navegación por niveles.

## 🧩 Proyectos integradores

1. **EDA de un dataset real** — describir, limpiar y visualizar datos con R base.

## 🛠️ Stack técnico

**HTML5 · CSS3 · JavaScript** (vanilla, sin frameworks) · [highlight.js](https://highlightjs.org/) para el resaltado de R · tipografías **Inter** + **JetBrains Mono**.

Cada idioma es una **SPA (Single Page Application) autocontenida en un solo archivo**: sin servidor ni dependencias que instalar. Se abre con doble clic, funciona sin conexión tras la primera carga y se despliega en cualquier hosting estático.

## 🚀 Cómo usarla

```bash
# Opción 1 — local: clona y abre con doble clic
git clone https://github.com/dmetrics1/learn-r.git

# Opción 2 — en vivo (ya publicada con GitHub Pages)
```
